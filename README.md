# Dychmon
### Prototyp systému na monitorovanie dýchania psov v celkovej anestézii

## O projekte
Dychmon je prototyp systému na monitorovanie a kontinuálne sledovanie dýchania psov. Projekt vznikol ako súčasť záverečnej práce s ieľom navrhnúť a implementovať cenovo dostupný systém na monitorovanie respiračnej aktivity pomocou elastického hrudného pásu so stretch senzorom. Signál je spracovávaný v reálnom čase s grafickou vizualizáciou.

## Hlavné vlastnosti
- kontinuálne monitorovanie respiračnej aktivity
- zobrazenie dychovej krivky v reálnom čase
- automatický výpočet dychovej frekvencie
- export nameraných dát do CSV
- modulárna architektúra programu
- podpora reálneho aj simulačného režimu

## Hardvér
- Respberyy Pi
- ADS1115 16-bit A/D prevodník
- stretch senzor
- elastický hrudný pás

## Softwér
Implementované v jazyky Python, knižnice:
- NumPy
- SciPy
- Matplotlib
- Tkinker
- Adafruit Blinka
- Adafruit CircuitPython ASx15
